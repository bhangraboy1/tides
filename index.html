This is the beginning of a project called TIDES - The Indian Dataset for Economic and Social sciences.

The key goal of this project is to create a data repository for publicly available data related to and focussed on India's economic development.

The project will seek to provide APIs written in Python and data visualizations. The data will be sourced from the Government of India (GOI), multi-lateral institutions such as the IMF, and other public sources. Initially, there will be little interpretation of the data and where data is manipulated or shown in a different way than it is produced we will try to cite sources for the justification for the metrics or representation. The data will be presented in a modestly flexible manner which should be useful for administrators in the Government of India.

If you have any suggestions for a dataset that should be included or a chart that should be shown - let the team know.
