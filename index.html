This is the beginning of a project called TIDES - The Indian Dataset for Economic and Social sciences. The key goal is to create a data repository that will be accessible via a Python API and to create curated visualization of data sourced the Government of India, multilateral institutions such as the IMF, and other freesource websites including Our World in Data. There will be limited interpretation of data, however, where this is done we will include citations to support why the choosen metrics or representation has been choose. The data will be presented in a modestly flexible manner which should be useful for administrators in the Government of India.
